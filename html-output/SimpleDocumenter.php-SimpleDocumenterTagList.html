SimpleDocumenterTagList

API Documentation

Tags

Description:
A SimpleDocumenterTagList holds a Reflection* object and an array of SimpleDocumenterTags. This object is an iterator, and provides methods like first(), count() and join(), which operate on the internal array of SimpleDocumenterTags. The method, tags(), returns the internal array.
